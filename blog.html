Základy digitálního marketingu

Pro všechny, kteří nechtějí usnout na vavřínech, nebo si plánují rozjet vlastní podnikání na internetu, společnost Google připravila skvělý on-line kurz “Základy digitálního marketingu”. Popisuje širokou škálu nástrojů, jakým způsobem postupovat, aby firma na internetu prorazila a dostala se tak do povědomí zákazníků. V úplných začátcích je třeba stanovit si cíl, čeho chcete u své firmy na internetu dosáhnout. Vedle vývoje webové stránky a jejího správného nastavení SEO optimalizace, byste rozhodně neměli minout registraci firmy v internetových adresářích. Zaregistrovat se můžete například na “Moje firma” na Googlu, nebo “Firmy.cz” na Seznamu. Je pak mnohem pravděpodobnější, že Vás potenciální zákazník objeví.

Další v pořadí, kde Vaši firmu můžete prezentovat, jsou sociální sítě. Je třeba uvědomit si, jaké platformy jsou k prezentaci Vaší firmy nejadekvátnější. Nemusíte se zaměřovat pouze na známé velké sociální sítě, jako je Facebook a LinkedIn, existuje spousta méně známých sociálních sítí, které jsou zaměřené na konkrétní úzký profil, například "Behance" pro kreativce, a právě tato platforma může skvěle zapadat do Vašeho profilu. Pokud tomu tak opravdu je, přihlášení do této sítě Vám může přivést spoustu potenciálních zákazníků.

V kurzu Vám dále vysvětlí na jakém principu funguje vyhledávání. Jaká klíčová slova tedy zvolit pro svou optimalizaci webu, například s pomocí nástroje plánovače klíčových slov od Googlu, který je dostupný po přihlášení do služby "Google Ads". Dále se dozvíte možnosti, pomocí jakých nástrojů můžete analyzovat výsledky zájmu o Vaši firmu. Například nástroj Google Search Console, dokáže poskytovat zpětnou vazbu, jak vidí Vaši webovou stránku ve výsledcích vyhledávání Google. Zaregistrujte se proto pro použití na “Web masters Google”.

A vůbec, zaregistrujte se na kurz! Stojí to za to!

Bootstrap

Bootstrap je jedním z nejpopulárnějších frameworků pro vývoj responzivních a mobilních webů. Pokud jsi začátečník, určitě Ti tento framework velmi pomůže. Pro využívání Bootstrapu je však nutné mít základy HTML a CSS.

V komponentách, které má bootstrap jsou jasně daná pravidla a umožňuje nám použít pouze to, co skutečně potřebujeme. Bootstrap nás vede a ukazuje směr jak dělat web správně. Zajistí nám také konzistentní zobrazení v prohlížeči a ušetří čas nad skládáním hotových komponent. Mnoho vývojářů podobné frameworky používá, mnoho je také odsuzuje. Kdo z nich má ale pravdu? Vyplatí se bootstrap používat? Odpověď je – vyzkoušej bootstrap! Až budeš mít zkušenost s několika projekty, pak se můžeš zařadit do jedné ze stran, která odsuzuje nebo chválí. Jelikož je ale Boostrap zdarma a open-source, určitě stojí za to si ho vyzkoušet!

Post 3

GIT je hit!

Pokud jste si kódování zamilovali, je načase vybrat si pro další vývoj vhodný editor.

Visual Studio Code (VS Code) se stal jedním z nejpopulárnějších editorů webových aplikací. V základu funguje jako obyčejný editor zdrojového kódu, s podporou syntaxe mnoha jazyků (HTML, Javascript, CSS..). Pokud se na něj ale podíváte blíže, uvidíte ladicí nástroj, podporu pro verzovací systémy a další vychytávky. Podporovány jsou také pluginy, které umí editor rozšířit o další formáty. VS Code je zdarma ke stažení na webu Visual Studia. Instalace je rychlá a jednoduchá, funguje na Windows, Mac OS X i Linuxu. Ke všemu obsahuje podporu pro Git a GitHub, který je nejen skvělým nástrojem pro vývoj Vašich projektů, ale zároveň je také jejich projekcí.

Git je verzovací systém, jehož účelem je ukládat historii projektu. Díky tomu se dá vracet do starších verzí, popřípadě na nich provádět nové změny. Zároveň je dohledatelné co a kdy jste do projektu přidali. Další přednost Gitu je možnost větvení projektu, které umožňuje pracovat na něčem, aniž byste riskovali, že se hlavní vývojová linie přeruší. A proč byste ho měli mít? Protože GIT je hit!

Post 2

Web krok za krokem!

Bez HTML a CSS se při tvorbě webu neobejdete. Jedná se totiž o základní stavební kameny webu. To, co vidíte na každé stránce, je tvořeno právě pomocí nich.

HTML (Hypertext Markup Language) je značkovací jazyk, kterým vytváříme obsah a strukturu stránky. Definujeme, který kus textu je nadpis, co je obyčejný odstavec, a kde na stránce například bude obrázek.
CSS (Cascading Style Sheet) je stylovací jazyk, kterým HTML stránce dáváme vzhled. Pomocí CSS nastavujeme, jak naše nadpisy vypadají, jakou mají barvu, jakým jsou písmem a velikostí. Dále například můžeme nastavit kolem textu rámeček, nebo přidat barvu pozadí. CSS zkrátka umí z obyčejné textové stránky udělat trendovou záležitost.

Než se rozhodnete pro některý z frameworků, který Vám tvorbu webové stránky usnadní, je zapotřebí pochopit základy HTML a CSS. Proto je důležité v počátku vytvořit projekt, kde všechny vlastnosti budou nadefinovány vlastnoručně.

Výborným pomocníkem pro ty, kteří se učí kódovat je online editor kódů - CodePen, který můžete využít bez jakýchkoliv instalací nástrojů, potřebných pro kódování. A to je v začátcích vážně skvělé! CodePen vám umožňuje psát kód do prohlížeče a vidět jeho výsledky při sestavování. Je také možné prohlížet práce ostatních registrovaných členů a nechat se tak inspirovat jejich tvorbou, protože je běžně využíván také vývojáři všech dovedností.
